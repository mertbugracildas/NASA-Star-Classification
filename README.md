## Project Overview

This project implements a machine learning model to classify stars based on their spectral characteristics (temperature, luminosity, radius, and absolute magnitude) using the NASA dataset.

The primary objective of this implementation is to demonstrate the mathematical foundations of **Logistic Regression** and **K-Means Clustering** algorithms by building them from scratch, without relying on high-level abstractions for the core logic. Additionally, a desktop graphical user interface (GUI) has been developed to provide real-time interaction with the model.

### Key Features

* **Data Preprocessing:** Automated cleaning, normalization, and feature scaling of raw astronomical data.
* **Algorithm Implementation:** Manual implementation of supervised (Logistic Regression) and unsupervised (K-Means) learning algorithms to ensure low-level control over the classification process.
* **Desktop Interface:** A functional GUI developed with `Tkinter` allowing users to input parameters and view classification results instantly.
* **Visualization:** Integrated plotting for data distribution and cluster analysis.

### Technologies

* **Core Language:** Python 3.x
* **GUI Framework:** Tkinter
* **Data Analysis:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn

# NASA Yıldız Sınıflandırma Projesi

## Proje Hakkında

Bu proje, NASA verisetini kullanarak yıldızları fiziksel özelliklerine (sıcaklık, parlaklık, yarıçap ve mutlak büyüklük) göre sınıflandıran bir makine öğrenmesi uygulamasıdır.

Çalışmanın temel amacı, **Lojistik Regresyon** ve **K-Means Kümeleme** gibi temel algoritmaları hazır kütüphane fonksiyonları yerine matematiksel temellerine dayanarak sıfırdan (from scratch) inşa etmektir. Geliştirilen model, kullanıcıların verileri anlık olarak analiz edebilmesi için bir masaüstü arayüzü (GUI) ile desteklenmiştir.

### Temel Özellikler

* **Veri Ön İşleme:** Ham verilerin temizlenmesi, normalizasyonu ve model için optimize edilmesi.
* **Algoritma Mimarisi:** Makine öğrenmesi algoritmalarının (Supervised & Unsupervised) manuel implementasyonu.
* **Etkileşimli Arayüz:** `Tkinter` ile geliştirilmiş, kullanıcı dostu veri giriş ve analiz ekranı.
* **Veri Görselleştirme:** Sınıflandırma sınırlarının ve veri kümelerinin grafiksel gösterimi.
